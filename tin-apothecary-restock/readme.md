# Tin Apothecary Restock Mode

## Installation
1. Upload the `tin-apothecary-restock` folder to `/wp-content/plugins/`.
2. Activate via the 'Plugins' menu in WordPress.

## Usage
1. Navigate to **Settings > Restock Mode**.
2. Toggle the checkbox to "On" to hide the store.
3. Update the text fields to inform customers of the next scent drop.
4. Logged-in Admins will still see the normal site for verification.

## Customization
CSS is located in `assets/restock-style.css` for further brand adjustments.
